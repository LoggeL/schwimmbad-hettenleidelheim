Allgemeine Hinweise

Diese Website ist eine statische Informationsseite. Es werden keine personenbezogenen Daten durch Formulare erfasst, keine Tracking‑Cookies gesetzt und keine Analysedienste eingesetzt.

Server‑Logfiles

Beim Aufruf der Seite können technische Informationen (z. B. IP‑Adresse, Datum/Uhrzeit, abgerufene Datei) vom Hoster in Server‑Logfiles gespeichert werden. Diese Daten sind für die Bereitstellung der Website erforderlich und werden nach den Vorgaben des Hosters gelöscht.

Externe Inhalte

Auf der Seite Anfahrt & Kontakt können Sie eine Karte von Google Maps aktiv laden. Erst mit Klick auf „Karte laden (Google)“ wird die Karte eingebunden und Daten an Google übertragen. Weitere Informationen dazu finden Sie bei Google unter Datenschutzerklärung.
